# 在庫管理システム

## 概要
このシステムでは、店舗で扱う商品の在庫管理を行うことができます。
商品の新規登録、編集、削除を行うことができます。

## 主な機能
-ログイン、ログアウト機能  
-商品一覧画面  
-商品新規登録、編集、削除機能  
-商品在庫管理機能  
-商品検索機能

## 開発環境
PHP 8.0.25  
Laravel 8.83.3
